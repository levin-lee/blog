A=B A等于B就返回true，适用于各种基本类型
A<=>B 都为Null则返回True，其他和=一样
A<>B 不等于
A!=B 不等于
AB 大于
A>=B 大于等于
A Between B And C 筛选A的值处于B和C之间
A Not Between B And C 筛选A的值不处于B和C之间
A Is NULL 筛选A是NULL的
A Is Not NULL 筛选A值不是NULL的
